--- layout: software category: software title: String Theory languages: Max MSP description: String is controller used to generate waveforms, curves, and envelopes using a camera, coloured string, and Max/MSP. asset: string_theory yt: sUz0lPUs1zs gh: String-Theory date: 2014-12-15 14:40:45 --- String is controller used to generate waveforms, curves, and envelopes using a camera, coloured string, and Max/MSP. Users draw curves representing objects such as a filter envelope using coloured string. The coloured curve is then captured by a camera and deciphered into a digital curve to be rendered out to audio by Max/MSP.
